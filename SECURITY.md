# Security Policy

If you discover a security vulnerability, please report it privately.

## Reporting

- Open a private security advisory in GitHub Security Advisories when possible.
- If advisories are not available, open an issue and mark it clearly as security-related without including exploit details.

## Scope

This repository includes MCP server tooling for browser capture and API analysis. Please include:

- Affected package(s)
- Reproduction steps
- Impact assessment
- Suggested remediation (if available)

## Response

We will acknowledge reports as quickly as possible and coordinate remediation and disclosure.

For additional package-level details, see `packages/cli/SECURITY.md`.
